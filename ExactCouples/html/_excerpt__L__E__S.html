ExactCouples :: excerptLES

excerptLES -- displays a few entries of a long exact sequence

Synopsis

  • Usage:
    excerptLES(k, l, M)
  • Inputs:
    • M, a module, over a couple ring, encoding a long exact sequence
    • k, an integer, the first cohomological degree to be examined
    • l, an integer, the last (and if omitted, l=k)
  • Outputs:
    • a net, depicting rows k..l of the long exact sequence, as well as the entries just before and after this row. There is also some ASCII art showing the snaking paths of the connecting maps.

Description

The encoding of a long exact sequence as a module is described in detail in longExactSequence
