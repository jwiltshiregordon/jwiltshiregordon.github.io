ExactCouples :: TorLES

TorLES -- the long exact sequence in Tor induced by an inclusion in the second coordinate

Synopsis

  • Usage:
    TorLES(k,W,X,A)
  • Inputs:
    • k, an integer, number of rotations of the long exact sequence to display
    • W, a module, giving a functor Tor(W,-)
    • X, a module
    • A, a module, a submodule of X
  • Outputs:
    • a net, showing the long exact sequence in Tor

Description

The long exact sequence is returned as a Net with the following general format:

|   .- ->      0
|  (
|   \
|
|                                                                   \
|   .- ->    W ** X    - - ->    W ** X/A   - - ->       0       - -'
|  (
|   \
|
|                                                                   \
|   .- ->  Tor_1(W,X)  - - ->  Tor_1(W,X/A)  - - ->    W ** A    - -'
|  (
|   \
|
|                                                                   \
|   .- ->  Tor_2(W,X)  - - ->  Tor_2(W,X/A)  - - ->  Tor_1(W,A)  - -'
|  (
|   \
|
|                                                                   \
|                          (k-3) more rows appearing    ...      - -'

Caveat

For computational access to the maps in the sequence, use TorCouple instead.
