ExactCouples :: coupleRing

coupleRing -- builds a couple ring

Synopsis

  • Usage:
    coupleRing(R,r,e,f)
  • Inputs:
    • R, a ring, the coefficients
    • r, an integer, the page number
    • e, a symbol
    • f, a symbol
  • Optional inputs:
    • Degrees (missing documentation) => ..., default value {{1, 0}, {-2, 2}}
  • Outputs:
    • a ring, R[e_r,f_r,Degrees=>{{1,0},{-2,2}}]

Description

These default degrees are chosen to meet the conditions of expectCoupleRing. Other degrees are possible using the optional argument Degrees or by building the couple ring by hand directly.

i1 : Q = coupleRing(QQ,7,e,f)

o1 = Q

o1 : PolynomialRing
 
i2 : describe Q

o2 = QQ[e , f , Degrees => {{1}, {-2}}, Heft => {1..2}, MonomialOrder => {MonomialSize => 32}, DegreeRank => 2]
         7   7              {0}  {2 }                                    {GRevLex => {1..2} }
                                                                         {Position => Up    }
 
i3 : expectCoupleRing Q

See also

  • expectCoupleRing -- accepts certain rings of the form R[e_r,f_r], and installs Page, isEvenDegree, and isOddDegree

Ways to use coupleRing:

  • coupleRing(Ring,ZZ,Symbol,Symbol)

For the programmer

The object coupleRing is a method function with options.
