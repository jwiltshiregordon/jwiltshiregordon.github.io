ExactCouples :: covariantExtLES

covariantExtLES -- the long exact sequence in Ext induced by an inclusion in the last coordinate of Hom

Synopsis

  • Usage:
    covariantExtLES(k,W,X,A)
  • Inputs:
    • k, an integer, number of rotations of the long exact sequence to display
    • W, a module, giving a hom-functor Hom(W,-)
    • X, a module
    • A, a module, a submodule of X
  • Outputs:
    • a net, showing the long exact sequence in Ext

Description

The long exact sequence is returned as a Net with the following general format:

|   .- ->        ...    (k-3) more rows appearing
|  (
|   \
|
|                                                                   \
|   .- ->  Ext^1(W,X)  - - ->  Ext^1(W,X/A)  - - ->  Ext^2(W,A)  - -'
|  (
|   \
|
|                                                                   \
|   .- ->   Hom(W,X)   - - ->   Hom(W,X/A)   - - ->  Ext^1(W,A)  - -'
|  (
|   \
|
|                                                                   \
|   .- ->      0       - - ->        0       - - ->   Hom(W,A)   - -'
|  (
|   \
|
|                                                                   \
|                                                        0       - -'

Caveat

For computational access to the maps in the sequence, use covariantExtCouple instead.

See also

  • excerptLES -- displays a few entries of a long exact sequence
  • covariantExtCouple -- the exact couple obtained by applying Ext(W,-) to a filtered module

Ways to use covariantExtLES:

  • covariantExtLES(ZZ,Module,Module,Module)

For the programmer

The object covariantExtLES is a method function.
