ExactCouples :: oneEntry

oneEntry -- builds a one-by-one matrix

Synopsis

  • Usage:
    oneEntry(row,col,ent)
  • Inputs:
    • ent, a ring element, drawn from some ring R
    • row, a list, a degree for the ring R
    • col, a list, a degree for the ring R
  • Outputs:
    • a matrix, a single-entry matrix with row degree row, column degree col, and entry ent

Description

i1 : R = QQ[x]

o1 = R

o1 : PolynomialRing
 
i2 : oneEntry({3},{5},x^2)

o2 = {3} | x2 |

             1       1
o2 : Matrix R  <--- R
 
i3 : oneEntry(,{5},x^2)

o3 = {3} | x2 |

             1       1
o3 : Matrix R  <--- R
 
i4 : oneEntry({3},,x^2)

o4 = {3} | x2 |

             1       1
o4 : Matrix R  <--- R

If integers are used in place of either row or col, they are multiplied by degree(ent).

i5 : S = QQ[y,Degrees=>{{4,-5}}]

o5 = S

o5 : PolynomialRing
 
i6 : oneEntry(3,,y)

o6 = {12, -15} | y |

             1       1
o6 : Matrix S  <--- S

Caveat

the ring element ent should be homogeneous

Ways to use oneEntry:

  • oneEntry(List,List,RingElement)
  • oneEntry(List,Nothing,RingElement)
  • oneEntry(Nothing,List,RingElement)
  • oneEntry(Nothing,ZZ,RingElement)
  • oneEntry(ZZ,Nothing,RingElement)
  • oneEntry(ZZ,ZZ,RingElement)

For the programmer

The object oneEntry is a method function.
