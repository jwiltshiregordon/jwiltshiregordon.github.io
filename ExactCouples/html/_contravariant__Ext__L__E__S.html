ExactCouples :: contravariantExtLES

contravariantExtLES -- the long exact sequence in Ext induced by an inclusion in the first coordinate of Hom

Synopsis

  • Usage:
    contravariantExtLES(k,X,A,Y)
  • Inputs:
    • k, an integer, number of rotations of the long exact sequence to display
    • X, a module
    • A, a module, a submodule of X
    • Y, a module, giving a hom-functor Hom(-,Y)
  • Outputs:
    • a net, showing the long exact sequence in Ext

Description

The long exact sequence is returned as a Net with the following general format:

|   .- ->        ...    (k-3) more rows appearing
|  (
|   \
|
|                                                                   \
|   .- ->  Ext^1(X,Y)  - - ->  Ext^1(A,Y)  - - ->  Ext^2(X/A,Y)  - -'
|  (
|   \
|
|                                                                   \
|   .- ->   Hom(X,Y)   - - ->   Hom(A,Y)   - - ->  Ext^1(X/A,Y)  - -'
|  (
|   \
|
|                                                                   \
|   .- ->      0       - - ->       0      - - ->   Hom(X/A,Y)   - -'
|  (
|   \
|
|                                                                   \
|                                                        0       - -'

Caveat

For computational access to the maps in the sequence, use contravariantExtCouple instead.

See also

  • excerptLES -- displays a few entries of a long exact sequence
  • contravariantExtCouple -- the exact couple obtained by applying Ext(-,Y) to a filtered module

Ways to use contravariantExtLES:

  • contravariantExtLES(ZZ,Module,Module,Module)

For the programmer

The object contravariantExtLES is a method function.
